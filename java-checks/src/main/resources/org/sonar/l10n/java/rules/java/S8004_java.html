Avoid empty finally block.

Noncompliant Code Example

// Noncompliant
public void testWithEmptyFinallyBlock() {
  try{
    System.out.println("hello");
  } catch (IOException e) {
    e.printStackTrace();
  } finally {

  }
}

Compliant Solution

public void test2() {
  try{
    System.out.println("hello");
  } catch (IOException e) {
    e.printStackTrace();
  } finally {
    return;
  }
}
