Method should have matched @Exception Tag Javadoc comment...

Noncompliant Code Example

// Noncompliant
/**
 * 测试没有throws标签
 */
public void testWithOutTag() throws IOException {
  System.out.println("hello");
}

Compliant Solution

/**
 *
 * @throws IOException IOException throw IOException
 */
public void testWithTag() throws IOException {
  System.out.println("hello");
}
